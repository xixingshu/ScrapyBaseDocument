# ScrapyDaseDocument

Scrapy 和 scrapy-redis 的 `start_requests` 对比分析 + 爬虫项目示例。

## 内容

| 文件 | 说明 |
|------|------|
| `start_requests对比分析.md` | Scrapy vs scrapy-redis `start_requests` 源码详解、对比表、配置项及扩展重写方法 |
| `问答记录.md` | 从 yield、isinstance 到 start_requests 的完整问答过程记录 |
| `Scrapy_spider/` | Scrapy 爬虫项目 |

## 环境

- Python 3.9 (conda env: `spider_base`)
- Scrapy 2.13+
- scrapy-redis 0.9.1
